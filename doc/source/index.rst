sekupy package
==============

Yet another neuroimaging package!

This package aims at easily integrating and running complex
pipelines for brain data analysis.


Examples Gallery
----------------

The examples gallery provides a set of runnable scripts illustrating the
main features of **sekupy**, from data loading to multivariate analysis and
result visualisation.
